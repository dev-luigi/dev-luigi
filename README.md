# Hi, I'm Luigi

Computer Science student at the University of Salerno, based in Nola (Naples), Italy.
I build websites, break things on purpose to understand how they work, and spend a fair amount of time on servers and home labs.

I run a small studio called **ImprontaWeb**, where I help professionals and small businesses get online with custom websites and e-commerce stores instead of cookie-cutter templates.

---

## What I'm into

I started coding because I wanted to build the kind of tools I couldn't find. Now I split my time between three things:

- **Web development.** Mostly full-stack: front-end with HTML, CSS, JavaScript and TypeScript; back-end in PHP, Java, Python and Node. WordPress when the project calls for it.
- **Systems & infrastructure.** I like making servers behave. I run my own home lab on Proxmox and VMware, mess around with Cisco networking, and treat virtualization like a hobby that pays off.
- **Hardware repair.** Phones, tablets, laptops, the occasional weird DIY electronics project. If it has a board inside, I want to open it.

On the side I'm a certified EASA A1-A3 drone pilot, with a bunch of hours in flight simulators before I touched a real one.
